# A React/Redux Boilerplate

> stater project using React/Redux

To use:

* Fork this repo
* Rename your repo to be the app you're building

```sh
git clone https://github.com/[your-username]/[your-app].git
cd [your-app] && yarn
```
This is the structure of the boilerplate:

```sh
.
├── client
│   ├── actions
│   │   └── index.js
│   ├── components
│   │   ├── AddWord.jsx
│   │   ├── App.jsx
│   │   ├── Word.jsx
│   │   └── Words.jsx
│   ├── reducers
│   │   ├── index.js
│   │   └── words.js
│   └── index.js
├── public
│   ├── index.html
│   └── main.css
├── server
│   ├── index.js
│   └── server.js
├── .gitignore
├── package.json
├── README.md
└── webpack.config.js
```
